# Open Business AI Agents 

Open-source AI agents for marketing, SEO, websites and small business automation.

## What is this project?

Open Business AI Agents is a collection of practical AI agents designed for real business tasks: marketing strategy, SEO analysis, website audits, advertising review, content planning and customer communication.

The project focuses on small businesses, consultants, marketers, founders and agencies that want to use AI agents in daily work without building everything from scratch.

## Why this project exists

Most AI agent examples are built for developers. This project is built around business workflows.

The goal is to provide reusable agent templates, prompts, workflows and examples that help people turn business expertise into structured AI-powered processes.

## Planned agents

- Marketing Strategy Agent
- SEO Agent
- Website Audit Agent
- Yandex Direct Analyst Agent
- Copywriting Agent
- Sales Assistant Agent
- Customer Support Agent
- Business Tracker Agent

## First milestone

The first version will include a Marketing Strategy Agent that helps analyze a product, define the target audience, clarify customer jobs-to-be-done, identify pains and objections, build an offer and suggest a website structure.
